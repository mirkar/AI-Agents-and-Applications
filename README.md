# AI-Agents-and-Applications
Based on Oreilly book AI-Agents-and-Applications

## Running chapters examples

Inside a chapter directory (like ch01) run 

Create virtual environment venv
```shell
 python -m venv venv
```

Activate virtual environment
```shell
.\venv\Scripts\activate
```

Upgrade pip
```shell
python.exe -m pip install --upgrade pip
```

Install required libriries
```shell
pip install -r requirements.txt 
```

Start Jupyter server
```shell
jupyter notebook
```

Chaptre's notebook(s) will be available at http://localhost:8888/tree
